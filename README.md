# Smart File Organizer

Smart File Organizer is a Python-based command-line tool that automatically organizes files into structured folders based on their file types. It is designed to be simple, extensible, and safe to use with features like dry-run mode and undo support.

---

## Features

- Organizes files into categories such as Images, Documents, Videos, Audio, and Code
- Supports recursive organization of nested folders
- Dry-run mode to preview changes before applying them
- Logging of file operations
- User-configurable file categories via a JSON file
- Undo functionality to revert the last operation
- Basic error handling for safer file operations
